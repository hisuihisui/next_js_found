## NextJS プロジェクト作成
### create-next-app のインストール
```
npm i -g create-next-app
```
※g：PC全体にインストール
### プロジェクトの作成
```
create-next-app [プロジェクト名]
```
### プロジェクトの起動
```
npm run dev
```
